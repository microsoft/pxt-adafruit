# Led

Control of the LED screen.

```cards
led.plot(0, 0);
led.unplot(0, 0);
led.point(0, 0);
led.toggle(0, 0);
led.brightness();
led.setBrightness(255);
led.stopAnimation();
led.plotBarGraph(0, 0);
led.fadeIn();
led.fadeOut();
led.plotAll();
led.screenshot();
led.toggleAll();
led.setDisplayMode(DisplayMode.BackAndWhite);
led.enable(false)
```
