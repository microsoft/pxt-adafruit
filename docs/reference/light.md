# Light

Functions to operate NeoPixel strips.

```cards
light.rainbowCycleAnimation();
light.runningLightsAnimation();
light.theatreChaseAnimation();
light.cometAnimation();
light.sparkleAnimation();
light.colorWipeAnimation(0);
light.createNeoPixelStrip(NeoPixelMode.RGB);
light.rgb(255, 255, 255);
light.hsv(255, 255, 255);
light.colors(NeoPixelColors.Red);
light.colorWheel(0);
light.statusLED(false);
light.NeoPixelStrip;
```
